# 基础语法

本篇文章，你将学会 Markdown 的基础语法，这些语法基本可以满足你日常文章书写要求。

---

## 标题

支持 `<h1>` ~ `<h6>` 共 6 个等级。

```markdown
# 等级 1

## 等级 2

### 等级 3

#### 等级 4

##### 等级 5

###### 等级 6
```

  <summary>效果预览</summary>

<Example>

</Example>

</details>

## 文本
